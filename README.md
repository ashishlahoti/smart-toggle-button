# \<smart-toggle-button>

Web Component

## Installation
```bash
npm i @aklahoti/smart-toggle-button
```

## Usage
```html
<script type="module">
  import '@aklahoti/smart-toggle-button';
</script>

<smart-toggle-button></smart-toggle-button>
```

## Start development
```bash
npm run start
```

## Publish your component
Bump your version first then release.
```bash
npm version patch
npm run publish 
```
